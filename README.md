# 📦 ExLoader

ExLoader — библиотека читов и игровых модификаций для CS2, Roblox, Minecraft и других игр.  
[Официальный сайт](https://exloader.xyz/)  

> ⚠️ Использование читов может привести к блокировке аккаунта в играх.
---

## 🚀 Что такое ExLoader?

ExLoader позволяет:  

- Скачивать различные моды (WH, Aimbot, SkinChanger и др.)  
- Запускать моды без стороннего инжектора  
- Использовать библиотеку более 200+ модификаций для популярных игр
